Beginning Each Sprint

At the beginning of each sprint, we hold a jam session and determine which stories to work on in the sprint. You determine the value of each story, and we jointly decide which stories to include in the sprint. We commit to what we can deliver during the sprint.

Ending Each Sprint

At the end of each sprint, we hold another jam session in which we measure the value added during the sprint. We use both objective measures (e.g. did we meet our commitment?) as well as subjective measures (e.g. is it "Good?"). {% endblock %} {% block sidebar %}
